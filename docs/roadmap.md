# Roadmap pratico

## Fase 1 - MVP local

- PWA com player, grelha, programas e playlists.
- Backend com dados seed e endpoints publicos.
- Modo demo quando nao existe stream configurado.
- Documentacao de arranque e checklist legal.

Estado: implementado.

## Fase 2 - Stream real

- Instalar AzuraCast ou usar fornecedor equivalente.
- Criar estacao Radium no painel de streaming.
- Configurar AutoDJ com conteudo autorizado.
- Copiar o URL publico do stream para `RADIUM_STREAM_URL`.
- Testar audio em desktop e mobile.

## Fase 3 - Operacao pessoal

- Substituir faixas placeholder por catalogo real licenciado/autorizado.
- Criar jingles, IDs e separadores.
- Ajustar `schedule.json` com a tua rotina.
- Ativar backups dos dados e assets.

## Fase 4 - Beta privado

- Partilhar so com um grupo pequeno.
- Medir estabilidade do stream.
- Registar problemas de player, volume, latencia e rede.
- Ajustar grelha e playlists com base no uso real.

## Fase 5 - Publico

- Fechar licenciamento e regulatorio antes de musica comercial.
- Publicar dominio final.
- Configurar HTTPS e reverse proxy.
- Preparar privacidade, cookies, termos e contacto.
